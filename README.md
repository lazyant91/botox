# Botox (보톡스) - AI 콘텐츠 필터 크롬 확장 프로그램

피드에 넘쳐나는 AI 봇(Bot)들의 유해한 독소(Tox) 같은 콘텐츠를 필터링하고 걸러내어, 인간의 오가닉한 콘텐츠만 깨끗하게 볼 수 있도록 돕는 센스 있고 가벼운 Chrome 확장 프로그램입니다.

---

## 📌 버전 정보 및 릴리즈 날짜
*   **현재 버전**: `v1.0.0` (Initial Release)
*   **릴리즈 날짜**: 2026년 6월 12일
*   **대상 플랫폼**: YouTube PC 브라우저 (Chrome)

---

## ✨ 핵심 기능 설명

### 1. 지능형 채널 차단 필터
*   유튜브 검색 결과 페이지 및 메인 피드에서 원치 않는 AI 채널의 동영상 노출을 완전히 차단합니다.
*   차단된 채널 목록에 있는 핸들 및 채널 고유 ID를 분석하여 해당 채널의 콘텐츠가 렌더링되기 전에 화면에서 부드럽게 숨김 처리합니다.

### 2. 고가속화된 수동 채널 등록 (타이핑 렉 0%)
*   수동으로 차단할 채널을 입력할 때 발생하던 실시간 파싱 병목을 없애기 위해, 타이핑 중에는 어떠한 지연도 주지 않습니다.
*   사용자가 **"차단" 버튼을 클릭하거나 엔터 키를 입력하는 시점**에만 비동기로 유튜브 oEmbed API 및 HTML 페치를 활용하여 일괄 유효성 분석 및 존재 여부를 검증합니다.

### 3. 유효성 검증 및 예외 차단
*   오탈자가 있는 잘못된 유튜브 도메인 주소나 존재하지 않는 유령 채널명을 수동 등록하려고 시도할 경우, 차단 등록을 즉시 중단하고 안내 토스트 메시지를 표시하여 올바르지 않은 필터 등록을 원천 차단합니다.
*   대소문자가 섞이거나 한글 주소 인코딩(`%EB%B8%94%EB%A3%B8_AI`) 상태로 입력하더라도 자동으로 정규화 및 한글 디코딩을 수행하여 중복 방지 로직과 정확하게 매칭됩니다.

### 4. 세련된 다크 테마 커스텀 스크롤바
*   차단 목록이 길어질 경우 생기는 스크롤바가 브라우저 기본 흰색 시스템 디자인으로 나타나 화면을 해치지 않도록, 확장 프로그램 팝업 및 내부 UI 요소에 다크 테마와 매끄럽게 어우러지는 반투명 슬림 스크롤바 디자인을 제공합니다.

## 🛠 설치 방법
1. 본 레포지토리를 클론하거나 ZIP 파일로 다운로드합니다.
2. 크롬 브라우저에서 `chrome://extensions/` 주소로 이동합니다.
3. 우측 상단의 **'개발자 모드'**를 활성화합니다.
4. **'압축해제된 확장 프로그램을 로드합니다'** 버튼을 클릭한 뒤, 본 프로젝트 폴더를 선택하여 로드합니다.
