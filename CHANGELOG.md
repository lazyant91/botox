# 변경 이력 (Changelog)

이 프로젝트의 모든 주목할 만한 변경 사항은 이 파일에 기록됩니다. 이 문서는 [Keep a Changelog](https://keepachangelog.com/ko/1.0.0/)의 규격을 준수합니다.

## [1.1.0] - 2026-06-12
### 추가됨 (Added)
- **YouTube 쇼츠(Shorts) 차단 기능**: 검색 결과 페이지 및 메인 피드에서 쇼츠 묶음 섹션 및 개별 쇼츠 비디오 카드를 보이지 않도록 숨김 처리.
- **Shorts 차단 설정 토글**: 설정 팝업(`popup.html`) 내에 ON/OFF 토글 스위치를 탑재하여 사용자가 실시간으로 쇼츠 차단 여부를 제어하도록 기능 추가 (기본값: ON).
- **무한 로딩 폭주 방어 스태빌라이저**: 쇼츠 차단으로 인한 피드 세로 수축 시 브라우저가 API 추가 페치를 무한 연사하여 발생하는 렉 문제를 막기 위해, 가상 높이 보정 지지대(`.botox-feed-stabilized::after`) 및 로딩 센서 락 장치(`lockContinuation`) 추가.

### 변경됨 (Changed)
- **MutationObserver 최적화**: 매 이벤트마다 문서 전체를 순회하던 `querySelectorAll` 스캔 방식을 폐지하고, 신규 노드(`addedNodes`)만 분리 검사하여 CPU 점유율을 60fps 수준으로 대폭 완화.
- **O(1) 캐싱 레이어 전환**: DOM Attribute 조작(`data-*`) 대신 자바스크립트 메모리 비교용 `WeakSet` 객체 주소를 활용해 이미 검증된 카드를 건너뛰도록 캐시 효율 최적화.
- **SPA 라우팅 연동 단일화**: 유튜브의 SPA 페이지 전환 감지(`yt-navigate-finish`) 시 가시성 리셋 및 캐시 인스턴스 갱신을 단일 `reapplyFiltering()` 경로로 묶어 리프레시 정합성 확보.

## [1.0.0] - 2026-06-12
### 추가됨 (Added)
- **최초 릴리즈 (Initial Release)**: YouTube PC 브라우저용 AI 채널 필터 크롬 확장 프로그램 배포.
- **채널 수동/인페이지 차단**: 동영상 카드 더보기 메뉴 주입 및 팝업창 등록을 통한 차단 채널 관리 기능 제공.
- **지연 시간 제거**: 채널 타이핑 입력 시 실시간 감지 방식에서 등록 버튼 누르는 시점에 일괄 비동기 fetch 검증하는 구조로 개선(타이핑 렉 0%).
- **다크 테마 스크롤바**: 확장 프로그램 목록 창 및 설정 모달 내부 다크 모드 맞춤 슬림 둥근형 웹킷 스크롤바 디자인 적용.
